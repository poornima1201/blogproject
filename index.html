This blog post shows a few different types of content that’s supported and styled with Bootstrap. Basic typography, images, and code are all supported.

A blog is a discussion or informational website published on the world wide web consisting of discrete, often informal diary-style text entries.

Heading

The emergence and growth of blogs in the late 1990s coincided with the advent of web publishing tools that facilitated the posting of content by non-technical users who did not have much experience with HTML or computer programming

Sub-heading

Previously, a knowledge of such technologies as HTML and File Transfer Protocol had been required to publish content on the Web, and early Web users therefore tended to be hackers and computer enthusiasts. In the 2010s, the majority are interactive Web 2.0 websites, allowing visitors to leave online comments, and it is this interactivity that distinguishes them from other static websites.
